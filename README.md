# Soft Skills — plugin do método Soft Business

Plugin com as habilidades do método **Soft Business**: o Sócio IA **LEON** + posicionamento, conteúdo, funil, vendas, webinário, lançamento e treino. Instale **uma vez** e receba as atualizações **automaticamente**.

## Como instalar (no Claude Code)
Abra o **Claude Code**, cole as duas linhas abaixo — **uma de cada vez** — e aperte Enter:

```
/plugin marketplace add molinateston/soft
/plugin install soft-skills@soft-skills
```

Isso instala **todas as skills de uma vez**. Não precisa caçar menu nenhum.

**Pra receber as melhorias sozinho** (faz uma vez só): digite `/plugin` → aba **Marketplaces** → escolha **soft-skills** → ligue **Atualização automática**. Daí cada nova versão chega sozinha quando você abre o Claude Code.

> ℹ️ Pelo **app do Claude (claude.ai)** não dá: o app só instala plugins do diretório oficial da Anthropic, não de repositórios. Use o **Claude Code** — é lá que o Soft Skills mora.

📖 Guia ilustrado passo a passo: https://leonardomolina.com.br/softskills
