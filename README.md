# Soft Business — Plugin

Plugin de skills do método **Soft Business**. Instale uma vez no Claude e receba as atualizações automaticamente.

## Instalar no Claude Chat (claude.ai)
Personalizar → Plugins pessoais (+) → Criar plugin → **Adicionar marketplace** → *Adicionar de um repositório* → `molinateston/soft` → **Sincronizar** → instale o plugin **soft**.

## Instalar no Claude Code
```
/plugin marketplace add molinateston/soft
/plugin install soft@soft-business
```
